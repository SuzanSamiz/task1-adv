# 📚 Library Management System

A simple and interactive Library Management System built using **HTML, CSS, and TypeScript**.
This project allows users to manage books, search, filter, and track availability ✨

---

## 🚀 Features

* ➕ Add new books  (Modal)
* 🔍 Search by title or author
* 📂 Filter books by category
* 🔄 Toggle book availability (Available / Borrowed)
* 📊 Live statistics (Total / Available / Borrowed)
* 🎨 Modern and responsive UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Modern UI + Animations)
* TypeScript (OOP + DOM Manipulation)

---

project-folder/

## 🧠 Concepts Used

* Object-Oriented Programming (OOP)

* Event Handling
* Dynamic Rendering
